これはHappy Hacking Keyboard（初代，2）をPS/2-USB変換アダプタで使うため
のMac OS X用デバイスドライバです。ただし，ちゃんと動く変換アダプタは限
られています。

インストール方法など，詳しくは次のwebサイトを参照してください。

  http://ichiro.nnip.org/HHKPS2USBDriver/
